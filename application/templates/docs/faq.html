Find answers to commonly asked questions about SecureRate.

SecureRate is a security tool designed to protect your online activities. It does this by analyzing a website on various security parameters and providing you with a security score. SecureRate also provides you with recommendations on how to improve your security score.

You can download SecureRate from the download page and following the installation guide.

Currently, SecureRate is available only for Google Chrome and other Chromium-based browsers.

You can report bugs on the GitLab page.

SecureRate gives a rating based on the following parameters:
  • Phishing: Is a site a known phishing website?
  • Malicious: Is a site known as a malicious website?
  • Breached Sites: Has this site been breached in the past and had it's data stolen?
  • 2FA: Does this site support two factor authentication?
  • HTTPS: Does this site use HTTPS?
  • DNSSEC: Is this site making use of DNSSec?

To analyze these parameters SecureRate uses various API's that have been vetted for security concerns.
